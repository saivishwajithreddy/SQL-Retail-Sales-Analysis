# SQL Retail Sales Analysis

## Project Overview

This project analyzes retail sales data using MySQL Workbench. The objective is to perform data cleaning, exploratory data analysis (EDA), and answer business-related questions using SQL queries.

---

## Tools Used

- MySQL Workbench
- SQL

---

## Dataset

- Retail Sales Dataset
- Total Records: 1000

---

## Project Objectives

- Import CSV data into MySQL
- Clean and validate the dataset
- Perform exploratory data analysis (EDA)
- Analyze customer purchasing behavior
- Analyze product category performance
- Generate business insights using SQL

---
